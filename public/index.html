Documentation of my toys API

Hello I'm Avraham Yosef Kallner and this Documentation of my toys API I've written.

toys Routes

Get information about toys via a full API

Method URL Description
Get /toys?page=number Get a list of all the toys , present 10 results by page number
Get /toys?perPage=number Get X animals in each page
Get /toys?reverse=sort-key Reverse by sort key: yes/no
Get /toys?sort=key-word From which value to sort by
Get /toys/price?min=number&max=number Price range
Get /toys/category/:category-name Get all animals from the same category
Get /toys/search?s=keyword Search products by name or info

toys Actions

Add, remove and edit allowed only for registered users

Method URL Body Description
Post /toys
  1. name (required)
  2. info (required)
  3. category (required)
  4. img_url
  5. price (required)
 Add a new toy Must to be connected
Edit /toys/:id-key
  1. name (required)
  2. info (required)
  3. category (required)
  4. img_url
  5. price (required)
 Edit a new toy Must to be connected
Delete /toys/:id-key Delete a new toy Must to be connected

Users Actions

To Login action you must first register

Method URL Body Description
Post /users
  1. name (required)
  2. email (required)
  3. password (required)
 Register to the system
Post /users/login
  1. email (required)
  2. password (required)
 Login account Must registered first
get /users/myInfo
  1. email (required)
  2. password (required)
 show user details Must registered first
get /users/usersList
  1. email (required)
  2. password (required)
 show all users details must to be an admin
